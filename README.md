# Приложение для грузчиков

Android-приложение для управления заказами грузчиков.

## Описание

Приложение позволяет:
- **Диспетчерам**: создавать заказы, отслеживать их статус
- **Грузчикам**: просматривать доступные заказы, брать их в работу, завершать

## Возможности

- ✅ Две роли: Диспетчер и Грузчик
- ✅ Создание заказов с полями:
  - Адрес
  - Дата и время
  - Описание груза
  - Цена за час работы
- ✅ Локальное хранение данных (Room Database)
- ✅ Уведомления о новых заказах
- ✅ Статусы заказов: Доступен, Занят, Завершен, Отменен
- ✅ Современный Material Design 3 UI

## Технологии

- **Язык**: Kotlin
- **UI**: Jetpack Compose + Material Design 3
- **База данных**: Room
- **Архитектура**: MVVM
- **Асинхронность**: Coroutines + Flow
- **Навигация**: Compose Navigation

## Сборка проекта

### Требования

- Android Studio Hedgehog | 2023.1.1 или новее
- JDK 17
- Android SDK API 24+ (минимальная версия)
- Android SDK API 34 (целевая версия)

### Автоматическая сборка в GitHub

Проект настроен для автоматической сборки через GitHub Actions:
1. Запушьте код в GitHub
2. GitHub Actions автоматически соберет APK
3. Готовый APK будет доступен в разделе Actions → Artifacts

### Ручная сборка

#### В Android Studio:
1. Клонируйте репозиторий:
```bash
git clone <repository-url>
cd LoaderApp
```

2. Откройте проект в Android Studio

3. Дождитесь синхронизации Gradle

4. Подключите Android-устройство или запустите эмулятор

5. Нажмите Run (зелёная кнопка)

#### Через командную строку:
```bash
git clone <repository-url>
cd LoaderApp
chmod +x gradlew
./gradlew assembleDebug
```

APK будет в: `app/build/outputs/apk/debug/app-debug.apk`

## Использование

### Первый запуск

При первом запуске приложение попросит создать профиль:
1. Введите имя
2. Введите телефон
3. Выберите роль (Диспетчер или Грузчик)

### Для диспетчера

1. Нажмите кнопку "+" для создания заказа
2. Заполните все поля:
   - Адрес
   - Дату (формат: дд.мм.гггг)
   - Время (формат: чч:мм)
   - Описание груза
   - Цену за час
3. Нажмите "Создать"
4. Заказ появится в списке и грузчики получат уведомление

### Для грузчика

1. Во вкладке "Доступные" видны все свободные заказы
2. Нажмите "Взять заказ" чтобы принять его
3. Заказ переместится во вкладку "Мои заказы"
4. После выполнения нажмите "Завершить"
